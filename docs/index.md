# 📚 实验文档索引

欢迎来到 STM32 微机控制实验库文档中心。这里详细记录了各个实验的设计思路、硬件连接以及核心代码说明。

## 📝 实验列表

### 1. [ADC 综合实验](adc_experiment.md)
- 电压监测与串口打印
- 基于 ADC 的 LED 亮度调节
- 电位器控速的跑马灯

### 2. [中断与定时器实验](interrupt_experiment.md)
- 按键外部中断 (EXTI)
- 二进制秒表实现
- 定时器中断 (TIM) 变频控制
- 软件 PWM 与呼吸灯

### 3. [LCD 绘制与电机控制综合实验](lcd_motor_experiment.md)
- 几何图形填充算法实现 (三角形/圆形)
- 电机 20kHz 过采样转速测量技术
- 实时电机监控仪表盘

## 🛠️ 如何使用这些文档
1. **预习阶段**: 阅读对应文档的“实验目标”和“内容”，理解实验原理。
2. **开发阶段**: 参考“硬件资源”进行连线，参考“关键 API”编写代码。
3. **调试阶段**: 对照“实验现象”验证程序是否工作正常。
